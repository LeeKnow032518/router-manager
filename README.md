# Router Manager

**Router Manager** - REST API сервис для работы с роутерами

## Возможности
- Отправка команд роутеру или всем роутерам
- Сохранение команд в PostgreSQL
- Вывод команд роутеру по "router_id"
- Прием подтверждений ("ack") выполнения команд
